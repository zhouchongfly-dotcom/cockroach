# 🪳 蟑螂大作战 (Roach Slayer)

一个使用纯前端技术（HTML5 Canvas + JavaScript）开发的打蟑螂小游戏。采用**俯视角**视角，真实还原厨房打蟑螂场景！

## 🎮 游戏特色

### MVP版本（已实现）
- ✅ **俯视角厨房场景**：从上往下看的真实厨房视角
- ✅ **拖鞋道具**：用拖鞋拍打蟑螂，带拍打动画和冲击波效果
- ✅ **智能AI**：蟑螂会朝着墙缝、橱柜缝隙、冰箱后面逃跑
- ✅ **藏身点系统**：蟑螂逃进藏身点会损失生命
- ✅ **真实厨房**：包含橱柜、灶台、水槽、冰箱、餐桌、椅子等物品
- ✅ **关卡系统**：随着关卡提升，蟑螂速度和生成频率递增
- ✅ **分数系统**：击杀蟑螂获得分数，过关获得额外奖励
- ✅ **计时系统**：每关限时60秒
- ✅ **俯视角蟑螂**：6条腿、触角、分段外壳，爬行动画逼真
- ✅ **音效系统**：击中、未击中、逃跑三种音效
- ✅ **暂停/继续**：游戏中可随时暂停

### 后续计划（渐进式开发）
- 🔜 **金币系统**：通关获得金币
- 🔜 **皮肤商店**：使用金币解锁皮肤
- 🔜 **成就系统**：解锁各种成就徽章
- 🔜 **节日活动**：特殊节日限定主题

## 🚀 快速开始

### 方式1：本地运行
直接用浏览器打开 `index.html` 文件即可开始游戏！

### 方式2：使用本地服务器
```bash
# 使用Python（推荐）
cd cockroach-game
python -m http.server 8000

# 或使用Node.js
npx serve

# 然后在浏览器访问
# http://localhost:8000
```

### 方式3：在线部署到Vercel

1. **安装Vercel CLI**
```bash
npm install -g vercel
```

2. **部署到Vercel**
```bash
cd cockroach-game
vercel
```

3. **生产环境部署**
```bash
vercel --prod
```

或者直接通过Vercel网站部署：
1. 访问 [vercel.com](https://vercel.com)
2. 导入GitHub仓库（需先将代码推送到GitHub）
3. Vercel会自动识别并部署

## 🎯 游戏玩法

1. **开始游戏**：点击"开始游戏"按钮
2. **消灭蟑螂**：用鼠标点击屏幕上快速移动的蟑螂
3. **注意生命**：不要让蟑螂逃出屏幕，每逃跑一只损失1颗心
4. **挑战高分**：在限定时间内尽可能多地击杀蟑螂
5. **关卡提升**：完成当前关卡后自动进入下一关，难度递增

## ⌨️ 快捷键

- `ESC` - 暂停游戏

## 📊 游戏数据

- **画布尺寸**：900x700px（俯视角厨房）
- **初始生命值**：3
- **每关时间**：60秒
- **击杀得分**：10分/只
- **过关奖励**：50分
- **蟑螂大小**：35px
- **基础速度**：2px/帧
- **速度增长**：+0.4px/关卡
- **藏身点数量**：6个（墙缝、橱柜缝隙、冰箱后）

## 🔧 自定义配置

你可以在 `game.js` 中的 `CONFIG` 对象修改游戏参数：

```javascript
const CONFIG = {
    canvas: {
        width: 900,             // 画布宽度
        height: 700             // 画布高度
    },
    cockroach: {
        baseSize: 35,           // 蟑螂大小
        baseSpeed: 2,           // 基础速度
        speedIncrement: 0.4,    // 速度增长
        spawnRate: 2000,        // 生成间隔(毫秒)
        spawnRateDecrement: 100 // 每关减少的生成间隔
    },
    game: {
        initialLives: 3,        // 初始生命
        levelTime: 60,          // 关卡时间(秒)
        pointsPerKill: 10,      // 击杀得分
        levelPointsBonus: 50    // 过关奖励
    },
    hideSpots: [
        // 蟑螂藏身点配置（墙缝、橱柜缝隙等）
        { x: 30, y: 30, width: 80, height: 15, type: 'crack' },
        // ...更多藏身点
    ]
};
```
